
# 📊 Real-Time Yahoo Finance Dashboard

A powerful real-time dashboard built with **Python**, **Dash**, and **Plotly**, designed to track the performance of **stocks** (AAPL, TSLA, GOOG, MSFT) and **cryptocurrencies** (BTC, ETH). The dashboard calculates technical indicators like **RSI**, **MACD**, and supports **multi-timeframe analysis** with automated data fetching from Yahoo Finance.

> "For traders, analysts, investors, or fintech startups who need insights that go deeper than just green and red candles."

---

## 🔍 Project Highlights

- ✅ Real-time stock and crypto price tracking
- ✅ Technical indicator support (RSI, MACD)
- ✅ Multi-timeframe selection: 1D, 1W, 1M, 3M, etc.
- ✅ Dynamic Dash/Plotly charts
- ✅ Automated data fetching via `yfinance`
- ✅ Extensible architecture — easily plug in new assets or timeframes

---

## 🛠️ How This Was Built

This project wasn't slapped together — it was **strategically crafted** using:

- `Python` for backend logic
- `Dash` (by Plotly) to create a responsive, interactive UI
- `Pandas` and `yfinance` for financial data wrangling
- `Plotly` for advanced visualization
- Modular code architecture for easy feature expansion

All charts and indicators are **custom-coded** — no drag-n-drop tools here. This is raw Python power, baby.
